# Simple Python Calculator

This is a simple Python calculator that performs basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features

- Add two numbers
- Subtract one number from another
- Multiply two numbers
- Divide one number by another (with zero-division check)
- Text-based menu interface for user selection

## How to Use

1. Make sure you have Python installed (version 3.x recommended).
2. Run the script using the following command:

`bash
python calculator.py
